# Ayaanle Show - Business Profile Website (v3)

**Website-ka Ayaanle Show** waa bog ganacsi oo casri ah, oo soo bandhigaya shirkaddaada, adeegyada, iyo xiriirka.

---

## Faylasha ku jira
- `index.html` — bogga hore (Home, About, Services, Contact)  
- `css/style.css` — style casri ah oo firfircoon  
- `contact.php` — PHP handler oo fariinta u dira email-ka (jipo2410@gmail.com)  
- `assets/hero.jpg` — sawir banner hero placeholder  
- `assets/favicon.png` — favicon placeholder  
- `README.md` — faylkan

---

## Features
- **Hero banner** oo leh qoraal soo dhawaynaya:  
  *“Welcome to Ayaanle Show – Entertainment & Events”*  
- **Responsive design** — mobile & desktop-friendly  
- **Social media links**: Facebook, YouTube, Instagram  
- **Contact form** — fariinta waxay ku dirtaa email-kaaga  
- **Modern color scheme**: #ff6b6b (primary), #4ecdc4 (buttons), #f0f2f5 (background), #222 (text), #1a73e8 (links)

## Sida loo isticmaalo
1. **Upload faylasha** server-kaaga (cPanel, FTP, ama hosting kale).

3. **Beddel sawir hero** iyo **favicon** haddii aad rabto sawir gaar ah.  
4. **Edit links-ka social media** si ay u noqdaan kuwa saxda ah.
